## 使用 CloudBase AI 生成图片

### 代码示例

```javascript
// 云函数中使用 CloudBase AI 生成图片
const tcb = require('@cloudbase/node-sdk');
const app = tcb.init({ env: 'cloud1-5gsbvhr91440e872' });

exports.main = async (event, context) => {
  const ai = app.ai();
  
  // 生成海报图片
  const result = await ai.createImageModel('hunyuan-image').generateImage({
    model: 'hunyuan-image',
    prompt: event.prompt,  // 使用上面的 Prompt
    size: '1024x1024',
    version: 'v1.9',
    style: '写实',
    footnote: 'NFC卡片助手',  // 水印
  });
  
  return {
    imageUrl: result.data[0].url,
    revisedPrompt: result.data[0].revised_prompt
  };
};
```

### 生成参数说明

| 参数 | 值 | 说明 |
|------|-----|------|
| `model` | `hunyuan-image` | 混元图像模型 |
| `size` | `1024x1024` | 图片尺寸 |
| `version` | `v1.9` | 最新版本 |
| `style` | `写实` / `插画` / `3D` | 图片风格 |
| `revise` | `true` | 自动优化 Prompt |
| `footnote` | `NFC卡片助手` | 水印文字 |

## 设计注意事项

1. ✅ **保持像素风格统一** - 所有海报必须使用统一的 8-bit 像素美术风格
2. ✅ **粗黑描边和硬阴影** - 这是产品的核心视觉识别特征
3. ✅ **色彩不超过 3 种主色** - 避免画面过于花哨
4. ✅ **信息层级清晰** - 标题 > 主视觉 > 副标题 > 说明文字
5. ❌ **禁止毛玻璃** - 不符合像素风定位
6. ❌ **禁止模糊效果** - 所有元素边缘必须清晰
7. ❌ **禁止强渐变** - 优先使用纯色块
