# V2.0 版本归档说明

**版本定位**：本地音频阶段  
**历史来源**：提交 `7c81e23 feat: add local audio NFC flow and sync docs`

## 版本特征

- 产品名已统一为 `NFC卡片助手`
- 在原有 `应用 / 音乐 / 网页` 写入能力基础上，新增：
  - 本地音频上传
  - 我的文件
  - 自定义 `tabBar`
- 当时主线仍是：
  - `audio_files`
  - `audio-file-service`
  - 本地音频上传后写入 NFC

## 当时方案边界

- 只支持本地音频，不支持视频
- 仍未升级到当前 V3.0 的“播放页分风格”方案
- 尚未切换到当前的：
  - `media_files`
  - `media_shares`
- `play-web-service`

## 归档策略

- V2.0 作为中间演进版本，仅保留版本说明
- 为避免与 V3.0 当前主线产生重复和混淆，不再在 `docs/` 中保留一整套旧需求、旧技术、旧 UI 文件
- 如果后续需要追溯，可从 git 历史恢复当时文档

## 当前关系

- V2.0 未作为最终主线长期维护
- 当前所有开发、联调、发布统一参考 `V3.0`
