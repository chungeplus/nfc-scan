# V1.0 版本归档说明

**版本定位**：项目初始发布阶段  
**历史来源**：仓库早期文档记录与版本历史说明

## 版本特征

- 产品名仍处于 `NFC直达` 阶段
- 以基础 NFC 写卡能力为主
- 主要围绕：
  - 应用写入
  - 音乐写入
  - 网页写入
- 像素风 UI 规范已初步建立

## 已知历史信息

根据历史文档索引，V1.x 阶段主要包含：
- 项目主需求文档
- UI 设计规范
- UI 高保真稿
- 交互原型
- NFC 集成说明
