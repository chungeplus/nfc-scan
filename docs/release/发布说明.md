# NFC Scan 发布说明

## 1. 版本说明

- 项目名称：`NFC Scan`
- 发布类型：`小程序发布版`
- 当前状态：`已提交微信官方审核`
- 文档更新时间：`2026-04-20`

## 2. 当前版本能力

### 2.1 首页

- 首页保留 `应用专区`、`音乐专区`、`网页专区` 三个主入口。
- `开发者承诺` 弹窗改为仅首次进入首页时展示一次，用户确认后写入本地缓存。
- 设备不支持 `NFC` 时首页弹出阻断提示，并阻止继续进入写入流程。

### 2.2 写入应用

- 选择应用弹框支持 `Android` / `iOS` 两个平台统一展示。
- 弹框支持遮罩关闭、底部操作区可见、固定高度与内部滚动。
- `iOS` 平台当前仅保留选择与提示，不开放实际写入能力。
- 预置应用列表已清理为当前发布版内容。

### 2.3 写入音乐

- 当前仅支持 `网易云音乐` 与 `QQ 音乐` 分享链接解析。
- 已移除 `小红书` 相关入口、逻辑、文案与原型展示。
- 空输入、解析失败、网络异常均会阻止继续写入，并给出明确反馈。
- 分享链接解析兼容重定向响应头大小写差异。

### 2.4 写入网页

- 输入区域为多行输入，支持粘贴和手动输入。
- URL 校验采用兼容性更高的正则方案。
- 支持 `query`、`hash`、`port` 以及无协议地址自动补全 `https://`。

### 2.5 NFC 写入弹窗

- 状态流转覆盖 `等待`、`写入中`、`成功`、`失败`、`重试`。
- 增加重复发现保护，避免同一标签重复写入。
- 非 `NDEF` 标签与写入失败场景均有明确提示。

### 2.6 版本更新

- 已在全局接入小程序版本更新检测。
- 新版本准备完成后，用户可直接确认并重启应用完成更新。
