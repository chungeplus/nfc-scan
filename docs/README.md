## 📋 快速导航

### 当前开发任务

- 🆕 **录音NFC功能**：用户上传录音 → 云存储 → NFC写入H5链接 → 碰一碰播放

### 技术方案

- **存储方案**：微信云开发云存储（支持永久访问URL）
- **前端方案**：微信小程序 + 云托管H5播放页
- **NFC方案**：通过H5链接拉起（个人主体小程序限制）

### 视觉风格

- **风格名称**：纸张底像素风 8-bit 工具型 UI
- **设计原则**：纸张底 + 彩色像素块 + 粗黑描边 + 硬阴影
- **参考规范**：[UI设计规范稿](./design/UI设计规范.md)
