## 当前版本重点

`V2.0` 当前聚焦在原有写卡能力基础上的增量更新，主要包括：

- 保留原有 `应用 / 音乐 / 网页` 写入能力
- 新增 `写入本地音频`
- 新增 `我的文件`
- 新增自定义 `tabBar`
- 新增云函数 `audio-file-service`
- 新增数据库集合 `audio_files`

---

## 当前方案说明

### 本地音频方案

- 从微信会话选择音频文件
- 上传到云存储 `audio-files/`
- 通过云函数写入数据库记录
- 返回可访问链接
- 将链接写入 NFC

### 当前能力边界

- 当前不是小程序内录音
- 当前不是独立 H5 播放页方案
- 当前不支持视频
- 当前不支持多主题媒体播放页
