## 一、项目概述

### 1.1 项目背景

NFC直达是一款微信小程序，核心功能为将NFC标签写入各类数据（应用包名、音乐链接、网页URL等），实现"碰一碰"快速直达。用户只需将手机靠近NFC标签，即可自动跳转至目标应用/音乐/网页。

### 1.2 目标用户

- 需要管理NFC标签的企业或个人用户
- 希望快速分享应用/音乐/网页内容的用户
- 对复古像素风格有偏好的用户群体

### 1.3 项目范围

| 包含 | 不包含 |
|------|--------|
| 首页功能入口 | 视频写入功能（后期规划） |
| 应用包名写入 | 用户登录/注册系统 |
| 音乐链接解析写入 | 后台数据管理系统 |
| 网页URL写入 | 标签批量管理 |
| NFC写入交互反馈 | |

### 2.2 用户流程图

```
用户打开小程序
    ↓
首次进入 → 显示开发者承诺弹窗 → 用户点击"了解" → 关闭弹窗
    ↓
NFC设备检测 → 不支持 → 显示提示 → 流程结束
    ↓ 支持
显示首页（三个功能入口）
    ↓
┌───────────────────────────────────────┐
│ 点击"应用" → 进入应用写入页 → 填写包名 → 点击写入 → NFC写入流程 │
│ 点击"音乐" → 进入音乐写入页 → 粘贴链接 → 解析 → 点击写入 → NFC写入流程 │
│ 点击"网页" → 进入网页写入页 → 填写URL → 点击写入 → NFC写入流程 │
└───────────────────────────────────────┘
    ↓
NFC写入流程
    ↓
┌─────────────────────────────────────┐
│ 1. 显示"请将NFC标签贴近手机"          │
│ 2. 用户贴近 → 检测到标签              │
│ 3. 显示"写入中，请勿移动"            │
│ 4. 写入成功 → 显示成功提示 → 确定返回  │
│ 5. 写入失败 → 显示错误原因 → 重试/确定 │
└─────────────────────────────────────┘
```

### 2.4 异常处理

| 场景 | 处理方式 |
|------|----------|
| 手机不支持NFC | 弹窗提示"您的设备不支持NFC功能"，禁用写入按钮 |
| NFC写入失败 | 显示错误原因（读写失败/标签损坏/容量不足等），提供重试按钮 |
| 包名格式错误 | 输入框下方红色提示"包名格式不正确" |
| URL格式错误 | 输入框下方红色提示"请输入有效的网址" |
| 音乐链接解析失败 | Toast提示"暂不支持该链接格式" |
| 小程序存在新版本 | 弹出更新提示，用户确认后重启更新 |

## 三、UI/UX 设计规范

### 3.1 整体风格

**风格名称**：纸张底像素风 8-bit 工具型 UI

**设计原则**：
- 页面主体使用暖纸色底，外层场景色使用浅绿色承托，保持工具类页面清晰可读。
- 所有核心组件保留粗黑描边、硬阴影、像素切角和块面层级，避免普通表单页观感。
- 可在导航栏和首页主卡上使用少量硬切分层渐变，模拟纸张与像素块质感，不使用柔和渐变、模糊阴影或玻璃拟态。
- 首页、表单页、弹窗页统一采用 `纸张底 + 彩色功能块 + 黑色描边 + 硬阴影` 的视觉组合。

### 3.2 色彩规范

| 场景绿 | `#88AA67` | 页面外层背景、舞台底色 |

| 纸张辅底 | `#EFE5C6` | 输入框、浅色容器、次按钮底 |
| 主描边 | `#2D2D2D` | 全局边框、像素图标主线 |

| 橙色 | `#F0BF66` | 首页主卡 |

| 绿色 | `#B8D98B` | 成功态、确认态 |
| 深绿色 | `#6EA85E` | 主按钮 |
| 红色 | `#E86A58` | 失败态、告警态 |

### 3.3 字体规范

**字体族**：
```scss
$font-family-pixel: 'Courier New', Courier, monospace;
```

**字号体系**（单位：rpx）：
| 用途 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 页面标题 | 32rpx ~ 36rpx | 加粗 | 深色 / 白色（视导航主题而定） |
| 卡片标题 | 28rpx ~ 32rpx | 加粗 | `#2D2D2D` |
| 正文 | 24rpx | 正常 | `#2D2D2D` |
| 辅助文案 | 20rpx ~ 22rpx | 正常 | `#626262` |
| 按钮文字 | 28rpx | 加粗 | 白色 / 深色 |

### 3.4 间距规范

```
xs  ：8rpx
sm  ：16rpx
md  ：24rpx
lg  ：32rpx
xl  ：48rpx
xxl ：64rpx
```

### 3.5 边框与阴影

**像素边框**：
```scss
$border-width: 4rpx;
$border-width-sm: 3rpx;
$border-color: #2d2d2d;
```

**硬阴影**（用于卡片、按钮、弹窗）：
```scss
box-shadow: 4rpx 4rpx 0 rgba(0, 0, 0, 0.14);
```

### 3.6 组件规范

#### 3.6.1 卡片组件

- 背景以 `橙 / 紫 / 青 / 纸张底` 为主，不使用高饱和霓虹暗底。
- 边框：`4rpx solid #2D2D2D`
- 圆角：`4rpx ~ 8rpx`
- 阴影：右下硬阴影
- 支持像素切角、缺口、虚线内框等纸张装饰

#### 3.6.2 按钮组件

**主按钮**：
- 背景：`#6EA85E`
- 文字：白色
- 边框：`4rpx solid #2D2D2D`
- 按下效果：`transform: translate(4rpx, 4rpx)` + 阴影消失

**次要按钮**：
- 背景：`#EFD86C` 或 `#EFE5C6`
- 文字：`#2D2D2D`
- 边框：`3rpx solid #2D2D2D`
- 按下效果：轻微位移 + 阴影消失

#### 3.6.3 输入框组件

- 背景：`#EFE5C6`
- 边框：`3rpx solid #2D2D2D`
- 内阴影：`inset 2rpx 2rpx 0 rgba(0, 0, 0, 0.12)`
- 字体：等宽像素字体
- 错误提示：统一使用 `#BE4A3A`

#### 3.6.4 弹窗/遮罩

- 遮罩背景：`rgba(38, 45, 30, 0.34 ~ 0.88)`
- 面板背景：`#F4EDD6`
- 标题条优先黄色，错误弹窗使用红色
- 操作区按钮不超过两个

#### 3.6.5 Toast提示

- 背景：纸张底或主题色底
- 文字颜色：按主题区分 `info / success / warning / error`
- 边框：`4rpx solid #2D2D2D`
- 位置：页面中部偏上或正中

### 3.7 图标规范

**图标方案**：使用 pixel-icon 组件渲染像素图标

**图标规格**：16×16 像素网格，渲染时按需缩放

**内置图标清单**：

| 图标名 | 用途 | 颜色 |
|--------|------|------|
| app | 应用写入入口 | 深绿 |
| music | 音乐写入入口 | 紫色 |
| web | 网页写入入口 | 深蓝 |
| question | 客服入口 | 深色 |
| back | 返回按钮 | 深色 / 白色 |
| home | 首页按钮 | 深色 |
| check | 成功勾选 | 深绿 |
| error | 失败图标 | 深红 |
| nfc | NFC标签标识 | 深色 |
| phone | 设备提示 | 深色 |

### 3.8 动效规范

**原则**：
- 优先使用短时、跳帧感、硬切换动效
- 页面主交互不依赖复杂转场，重点强化按钮按压、底部弹层上滑、NFC 状态流转
- 动画时长建议 0.08s ~ 0.6s

**可用动画**：
| 动画名 | 效果 | 用途 |
|--------|------|------|
| slideUpFromBottom | 底部上滑 | 应用选择弹层 |
| pixelLoading | 步进旋转 | NFC 写入中 |
| phonePulse | 设备脉冲 | NFC 等待态 |
| tagApproach | 标签靠近 | NFC 等待态 |
| pressDown | 位移按压 | 按钮点击反馈 |

### 4.4 API调用

**NFC能力**：
```javascript
// 获取NFC适配器
const nfcAdapter = wx.getNFCAdapter();

// 监听标签发现
nfcAdapter.onDiscovered((res) => {
    // 获取NDEF实例
    const ndef = nfcAdapter.getNdef();
    // 写入数据
    ndef.writeNdefMessage({
        uri: 'https://example.com'
    });
});
```

**音乐解析**：
- 网易云音乐：`https://y.music.163.com/m/song/{id}`
- QQ音乐：`https://y.qq.com/n/ryqq/song/detail/{id}`
