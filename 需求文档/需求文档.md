## 一、项目概述

### 1.1 项目背景

NFC直达是一款微信小程序，核心功能为将NFC标签写入各类数据（应用包名、音乐链接、网页URL等），实现"碰一碰"快速直达。用户只需将手机靠近NFC标签，即可自动跳转至目标应用/音乐/网页。

### 1.2 目标用户

- 需要管理NFC标签的企业或个人用户
- 希望快速分享应用/音乐/网页内容的用户
- 对复古像素风格有偏好的用户群体

### 1.3 项目范围

| 包含 | 不包含 |
|------|--------|
| 首页功能入口 | 视频写入功能（后期规划） |
| 应用包名写入 | 用户登录/注册系统 |
| 音乐链接解析写入 | 后台数据管理系统 |
| 网页URL写入 | 标签批量管理 |
| NFC写入交互反馈 | |

## 二、功能需求

### 2.1 功能清单

| 编号 | 功能名称 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| F01 | 首页功能入口 | 展示应用写入/音乐写入/网页写入三个入口卡片 | P0 |
| F02 | 写入应用 | 输入Android应用包名，写入NFC标签 | P0 |
| F03 | 写入音乐 | 解析网易云/QQ音乐链接，写入NFC标签 | P0 |
| F04 | 写入网页 | 输入URL地址，写入NFC标签 | P0 |
| F05 | NFC设备检测 | 自动检测手机NFC是否可用 | P0 |
| F06 | NFC写入交互 | 发现标签→写入中→成功/失败 全流程反馈 | P0 |
| F07 | 开发者承诺弹窗 | 首次进入展示隐私声明 | P1 |
| F08 | 客服入口 | 悬浮按钮跳转微信客服 | P1 |
| F09 | 粘贴功能 | 从剪贴板快速粘贴链接/包名 | P1 |

## 三、技术方案

### 3.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | 原生小程序 | 不依赖第三方框架 |
| 组件 | 自研像素UI组件 | 全自定义，不依赖TDesign |
| 样式 | SCSS | 全局变量+混合器管理 |
| 图标 | pixel-icon组件 | 16×16像素网格渲染 |
| 字体 | 等宽字体 | Courier New / monospace |

### 3.2 项目结构

```
miniprogram/
├── components/               # 自定义组件
│   ├── pixel-navbar/       # 像素风导航栏
│   ├── pixel-toast/        # 像素风提示
│   ├── pixel-icon/         # 像素风图标
│   └── scan-dialog/        # NFC写入弹窗
├── pages/                   # 页面
│   ├── write-menu/         # 首页
│   ├── write-app/          # 应用写入
│   ├── write-music/        # 音乐写入
│   └── write-web/          # 网页写入
├── styles/                  # 全局样式
├── utils/                   # 工具函数
└── app.js
```

### 3.3 API调用

**NFC能力**：
```javascript
const nfcAdapter = wx.getNFCAdapter();
nfcAdapter.onDiscovered((res) => {
    const ndef = nfcAdapter.getNdef();
    ndef.writeNdefMessage({ uri: 'https://example.com' });
});
```

## 四、验收标准

- 首页三个功能入口正常显示，点击可跳转
